# JavaScript Card Tooltip

A custom tooltip component built with **HTML**, **CSS**, and **Vanilla JavaScript**. When users hover over the truncated text inside a card, a tooltip appears and displays the full content.

## Features

* Custom tooltip built with Vanilla JavaScript
* Display full text on hover
* Multi-card support
* Dynamic tooltip positioning
* Clean and responsive card layout
* Lightweight and easy to customize

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
